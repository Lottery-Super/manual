##### Environment setup

See .env.example file

##### Install packages

```sh
yarn
```

##### Start app development

```sh
yarn dev
```

##### Start app

```sh
yarn start
```

##### Start app (staging)

```sh
yarn staging
```

##### Start app (production)

```sh
yarn production
```
